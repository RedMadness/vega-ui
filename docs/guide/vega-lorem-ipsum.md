# Lorem Ipsum
Simple content placeholder.

## Usage
```html
<VegaLoremIpsum />
```

## Props

| Prop      | Type    | Default  | Description                   |
|-----------|---------|----------|-------------------------------|
| `font-size` | String | `20px`   | Set the font size of the content. |
| `padding`   | String | `0`      | Set the padding of the content.   |
